# Yazi file manager

This setup uses Yazi instead of Thunar.

Keybind:

```text
SUPER + E -> kitty -e yazi
```

Installed support packages:

- `yazi`
- `ueberzugpp`
- `ffmpeg`
- `ffmpegthumbnailer`
- `poppler`
- `resvg`
- `imagemagick`
- `chafa`
- `jq`
- `p7zip`
- `unzip`
- `unrar`
- `xdg-utils`

Config is restored into:

```text
~/.config/yazi
```
