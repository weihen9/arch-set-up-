# Package groups

## Core
See `pkg/core.txt`. Includes Arch base additions, AMD microcode, NetworkManager, sudo, git, docs, and pacman maintenance tools.

## Desktop shell
See `pkg/desktop-shell.txt`. Includes Hyprland-adjacent userland: kitty, alacritty, rofi, Waybar, awww, SwayNC, pywal, hyprpicker, fonts, themes, and icon packages.

## Filesystem / file manager
See `pkg/filesystem.txt`. This setup uses `yazi` instead of Thunar. It also includes preview/archive/mount helpers such as ueberzugpp, ffmpeg, poppler, imagemagick, chafa, gvfs, udiskie, and filesystem utilities.

## NVIDIA
See `pkg/gpu-nvidia.txt` or `pkg/gpu-nvidia-dkms.txt`.

## Gaming
See `pkg/gaming.txt`. Includes Steam, GameMode, MangoHud, Gamescope, Discord.

## Dev basics
See `pkg/dev-basic.txt`. Basic coding tools only; no ML/CUDA/PyTorch/Jupyter stack.

## AUR
See `pkg/aur.txt`. Includes LibreWolf, official VS Code binary, Vesktop, ProtonUp-Qt, and Qogir icons.
